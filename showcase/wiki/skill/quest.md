# Quest Skill

This showcase page documents the dedicated quest workflow as a skill-level
interface.

## Commands

- `/quests` suggests the active quest for the session.
- `/checkpoint` drafts the next checkpoint for the active quest.
- `/split` drafts a new quest name and handoff when the quest drifts.

## Rule

The quest is fixed once chosen. The agent does not sweep the wiki and pick work
autonomously.
